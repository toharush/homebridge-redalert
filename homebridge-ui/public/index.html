Adds a Switch to HomeKit that turns OFF when all sources are unreachable.
Consecutive failures per source before reporting unhealthy. Default: 5.
Webhooks
Send an HTTP POST when a sensor activates or deactivates. Payload includes event type, sensor name, city, and title.
